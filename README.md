# destboard

## Description

Output all patterns of anagram.

## Requirement

- Python 3.11.2

## Usage

Run script as below,

```
$ python ./main.py abc
abc
acb
bac
bca
cab
cba
```

## Install

Fork and clone this repository.

```
$ git clone git@github.com:[yourname]/anagram.git
```

Create python virtual env.

```
$ python3 -m venv .env
$ source ./.env/bin/activate
```

Install packages.

```
(.env)$ python -m pip install -r .\requirements.txt
```
